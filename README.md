# Сухомлин Артём

## Описание работы

Куб: для обработки физики используется CharacterController (так как не нужна глубокая проработка 
физики). Куб движется равноускоренно, переключение между полосами осуществляется с 
использованием MoveTowards, прыжок реализован изменением вертикальной скорости. При столкновении 
с препятствием отнимается соответственное количество здоровья. При достижении 0
игрок умирает, управление отключается, через 2 секунды сцена перезагружается

Управление: Клавиши A/D (стрелочки влево, вправо) переключают полосы, 
пробел отвечает за прыжок, 
используется новая система управления

Камера и плоскость: для них используется один и тот же скрипт для следования за кубом по оси z, 
отличается только offset (в инспекторе изменен), плавность реализована через функцию Lerp, 
параметр smoothSpeed характеризует плавность движения камеры, чем ниже, тем медленнее камера 
следует за кубом (он постепенно отдаляется с увеличением скорости). У 
плоскости все значения (x, y, z) должны быть равны 0 (в инспекторе)

Препятствия: генерируются перед игроком на расстоянии spawnDistance с интервалом spawnInterval 
на одной из трех линий. Для хранения урона и цвета используется Scriptable 
Object

Главная сцена - SampleScene (единственная), главные скрипты - CubeScript и 
Spawner

## Дополнительные функции

Используется новая система ввода, также реализовано ускорение игры с 
течением времени

# Home Assignment 2

## Персонаж

Вместо куба используется анимированная моделька робота

## Бонусы

1. Восстановление здоровья - синий шар, который восстанавливает HealthAmount
   (1) hp
2. Буст скорости - желтая капсула, которая увеличивает скорость в 
   SpeedMultiplier(1,5) раз на Duration(3) секунд. Если персонаж врезается 
   в препятствие, то бонус заканчивает своё действие

Все данные хранятся в ScriptableObject

## HUD

Каждый элемент худа - это объект TextMeshPro, который обновляется каждый 
кадр и принадлежит родительскому объекту Canvas. Рекорд сохраняется в 
PlayerPrefs (только по окончании забега). Если нынешнее 
количество очков 
больше рекорда, то соответсвующий текстовый элемент худа начинает 
обновляться вместе с обычным счетчиком очков.

После нажатия на клавишу "esc" игра ставится на паузу, у игрока появляются 
2 опции: выйти в главное меню и продолжить игру

## Главное меню

Сцена MainMenu, которая теперь является точкой входа программы, содержит 
элемент Canvas, являющийся родительским для остальных элементов меню:
1. Текст с названием игры
2. Текст с именем автора
3. Текст рекордом, который обновляется при каждом запуске меню
4. Кнопка запуска игры
5. Кнопка выхода из игры

## Анимации

При запуске сцены персонаж начинает идти, после чего переходит на бег (т.к. 
его скорость становиться > 0). По нажатию клавиши прыжок, запускается 
анимация прыжка, во время действия ускорителя робот переходит на спринт.
При столкновении с препятствием персонаж за 200 мс становиться красным 
полупрозрачным, потом за 1 с постепенно возвращает свой нормальный цвет. 
Такая же анимации при подборе орбы здоровья, но персонаж становится зеленым
