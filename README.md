# Сухомлин Артём

## Описание работы

Куб: для обработки физики используется CharacterController (так как не нужна глубокая проработка 
физики). Куб движется равноускоренно, переключение между полосами осуществляется с 
использованием MoveTowards, прыжок реализован изменением вертикальной скорости. При столкновении 
с препятствием отнимается соответственное количество здоровья. При достижении 0
игрок умирает, управление отключается, через 2 секунды сцена перезагружается

Управление: Клавиши A/D (стрелочки влево, вправо) переключают полосы, 
пробел отвечает за прыжок, 
используется новая система управления

Камера и плоскость: для них используется один и тот же скрипт для следования за кубом по оси z, 
отличается только offset (в инспекторе изменен), плавность реализована через функцию Lerp, 
параметр smoothSpeed характеризует плавность движения камеры, чем ниже, тем медленнее камера 
следует за кубом (он постепенно отдаляется с увеличением скорости). У 
плоскости все значения (x, y, z) должны быть равны 0 (в инспекторе)

Препятствия: генерируются перед игроком на расстоянии spawnDistance с интервалом spawnInterval 
на одной из трех линий. Для хранения урона и цвета используется Scriptable 
Object

Главная сцена - SampleScene (единственная), главные скрипты - CubeScript и 
Spawner
